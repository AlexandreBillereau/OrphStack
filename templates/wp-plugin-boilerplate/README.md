# Orphic WP Plugin Boilerplate

This repository serves as a boilerplate for initializing a customer website plugin. It provides a structured starting point for developing WordPress plugins with best practices in mind.

Clone this repository when initializing a customer website plugin.

## Dependencies

This plugin requires ACF PRO to function correctly. Ensure you have a valid ACF PRO license and include it in your plugin's dependencies.

## Utilisation du script de renommage

Pour renommer le plugin, suivez ces étapes :

1. Assurez-vous d'avoir Python 3 installé sur votre système.

2. Ouvrez un terminal et naviguez jusqu'à la racine du projet.

3. Exécutez le script `rename_plugin.py` avec la commande suivante :

   ```
   python3 rename_plugin.py
   ```

4. Le script parcourra tous les fichiers du projet, renommera les fichiers nécessaires et mettra à jour le contenu pour refléter le nouveau nom du plugin.

5. Une fois le script terminé, vérifiez les modifications et assurez-vous que tout fonctionne correctement. (installer ACF et regarder si les blocks fonctionnents)

Enjoy !
